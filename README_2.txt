To run this project I had to:

1.   Serve a web page and JSON file through a local server:

- On the VS Terminal run the code:
python -m http.server

2. Access the Server:

- Open a new browser and digit:
http://localhost:8000

3. INSTALL LIVE Server
- CLICK WITH THE RIGHT BUTTON ON THE HTML FILE, CHOOSE THE OPTION: 
    OPEN WITH THE LIVE SERVER
